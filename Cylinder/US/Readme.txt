17/12/2021 - Hung Truong, University of Strasbourg
Data of preliminary results for Test case: Mesh motion - Cyl of High Fidelity CFD workshop 2022
The data contains forces and moment solved with 5 meshes from coarse to fine (DB0 coarsest and DB4 finest)
There are four motions:
	Translation: files with prefix forces_T_*
	Rotation: files with prefix forces_R_*
	Deform: files with prefix forces_D1_*
	Composite: files with prefix forces_D2_*

Each file contains 5 columns of data as follow:
Iteration	Time		Fx/Drag		Fy/Lift		Cz/Moment
